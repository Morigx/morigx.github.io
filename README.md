# gtnntg.com - Цифровая Личность

Персональный сайт-визитка с динамическим статусом через Google Calendar.

## 🚀 Структура

- **Главная (`/`)** - Минималистичное лицо с ником `gtnntg`
- **Хаб карточек (`/digitalcard`)** - Центр навигации по цифровым карточкам
- **Цифровая карточка (`/digitalcard/gtnntg`)** - Персональное пространство с автообновляемым статусом

## ✨ Функции

### Автоматически
- 🌓 **Тема**: Определяется по настройкам браузера (тёмная по умолчанию)
- 🌐 **Язык**: Определяется по браузеру (RU/EN, резерв - русский)
- 💾 **Сохранение**: Выбор темы и языка сохраняется в localStorage

### Цифровая карточка
- 📅 **Google Calendar интеграция**: Статус обновляется автоматически на основе событий календаря
- 🔄 **Автообновление**: Каждую минуту проверяется активное событие
- 🎨 **Динамический контент**: Аватар, статус и ссылки меняются в зависимости от события

## 🛠 Настройка Google Calendar

1. Создайте публичный Google Calendar
2. Получите ICS-ссылку:
   - Откройте настройки календаря
   - Найдите "Secret address in iCal format"
   - Скопируйте ссылку

3. Откройте `/src/utils/calendarParser.js` и замените:
```javascript
const GOOGLE_CALENDAR_ICS_URL = 'YOUR_GOOGLE_CALENDAR_ICS_URL_HERE';
```
на вашу ICS-ссылку.

4. Создавайте события с JSON в описании:
```json
{
  "avatar": "https://example.com/avatar.jpg",
  "status": "В работе",
  "links": [
    {"name": "Проект", "url": "https://github.com/..."}
  ]
}
```

## 🚀 Развёртывание на GitHub Pages

1. Создайте репозиторий `gtnntg/gtnntg.github.io`
2. Запушьте код
3. Включите GitHub Pages в настройках репозитория
4. Для кастомного домена добавьте CNAME файл с `gtnntg.com`

## 🎨 Цветовая схема

**Тёмная тема (по умолчанию):**
- Фон: `#0a0a0a`
- Текст: `#f0f0f0`
- Градиент: Циан `#00ffff` → Фиолетовый `#9d4edd`

**Светлая тема:**
- Фон: `#ffffff`
- Текст: `#1a1a1a`
- Градиент: Океанский `#00d4ff` → Фиолетовый `#7b2cbf`

## 📁 Структура проекта

```
src/
├── components/
│   ├── Header.jsx       # Шапка с переключателями
│   └── Header.css
├── contexts/
│   ├── ThemeContext.jsx     # Управление темой
│   └── LanguageContext.jsx  # Управление языком
├── hooks/
│   └── useCalendarStatus.js # Хук для статуса из календаря
├── pages/
│   ├── Home.jsx/css     # Главная страница
│   ├── Hub.jsx/css      # Хаб карточек
│   └── Card.jsx/css     # Цифровая карточка
├── utils/
│   └── calendarParser.js # Парсинг ICS календаря
├── main.jsx             # Точка входа с роутингом
└── index.css            # Глобальные стили
```

## 🔧 Технологии

- **Vite + React** - Фреймворк и сборщик
- **React Router DOM** - Роутинг
- **ical.js** - Парсинг ICS файлов
- **CORS Proxy** - allorigins.win для доступа к Google Calendar
